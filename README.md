# Accurate-Cyber-Defense-RedScan-Bot
Accurate-Cyber-Defense-RedScan-Bot is a secure, ethical penetration testing tool that allows authorized users to scan networks, detect vulnerabilities, and generate actionable reports. 
The tool features seamless Telegram configuration, enabling users to send commands, monitor scans, and receive alerts remotely in real time.

## How to clone the repo
```bash
git clone https://github.com/Iankulani/Accurate-Cyber-Defense-RedScanner-Bot.git
cd Accurate-Cyber-Defense-RedScanner-Bot
```
## How to run
```bash
python3 Accurate-Cyber-Defense-RedScanner-Bot.py
```
